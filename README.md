# speech-transcription-and-summarization
A real-time speech-to-text and AI-based summarization application that supports English and Malayalam.
The app captures live audio, converts it into text, detects English questions automatically, and generates a concise summary
Built using Streamlit, SpeechRecognition, and Hugging Face Transformers, this project demonstrates practical AI integration in a user-friendly application.


Features:
-Live Speech Transcription

-Supports English and Malayalam

-Continuous recording until the speaker stops

-Automatic Question Detection

-Adds question marks for English questions intelligently

- Transcript Display
-Uses Hugging Face summarization models
